CLOUDFORMATION AWS RDS PROVISONING:
Please provide Cloudformation base template, and the following parameters (separated by new line):

Key: DBName
Description: The name of RDS Database.

Key: DBUser
Description: The username of RDS db.

Key: DBPassword
Description: The password of RDS db.

Key: DBAllocatedStorage
Description: The size of the RDS db in GB.

Key: DBInstanceClass
Description: The DB instance class determines the computation and memory capacity of an Amazon RDS DB instance.

Key: MultiAZ
Description: Takes only TRUE/FALSE value. If True, AWS would create automatic RDS failover.
